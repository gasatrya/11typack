# 11typack

Build your next website faster with 11typack. **11typack** is a starter kit to help you create a website using modern & standard technologies like [Eleventy](https://www.11ty.dev/), [Webpack](https://webpack.js.org/), [Babel](https://babeljs.io/) and [Tailwind CSS](https://tailwindcss.com/), while also considering ease of use, performance and browser compatibility.
